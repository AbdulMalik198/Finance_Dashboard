#  Personal Finance Dashboard

A responsive web application that helps users track income and expenses, visualise spending habits, and monitor financial balance in real time.

This project demonstrates practical front-end development skills including DOM manipulation, data persistence, chart visualisation, and user input validation.

---

##  Live Demo
https://statuesque-starlight-f1110a.netlify.app/

---

##  Features

- Add income and expense records  
- Automatic calculation of totals and balance  
- Category-based tracking  
- Interactive pie chart (Chart.js)  
- CSV export functionality  
- LocalStorage data persistence  
- Fully responsive layout
